# Stripe 埋め込み決済を動かすための作業手順

LP（lp.well-c.biz/20260522b-s/v2/）内に Stripe のクレジットカード入力欄を**直接展開**するための、阿久津さんの手作業手順です。所要時間 約5〜10分。

---

## ステップ1：Stripe管理画面で「Secret Key（ライブ用）」をコピー

1. ブラウザで次のURLを開く：
   https://dashboard.stripe.com/apikeys

2. 画面右上のトグルが「**テストモード**」になっていたら、**オフ**にして「本番モード」に切り替える

3. 画面中央の「**標準キー**」または「**シークレットキー**」の欄に `sk_live_...` で始まる文字列がある
   - 「**キーを表示**」または「**Reveal live key**」を押すと全体が見える
   - 全体をコピー（Ctrl+C / Cmd+C）

---

## ステップ2：Stripe管理画面で「Price ID（2,980円の価格ID）」をコピー

1. ブラウザで次のURLを開く：
   https://dashboard.stripe.com/products

2. 「**勉強会アーカイブ動画＋PDF資料**」（または 2,980円 の商品）をクリック

3. 商品ページの下のほうにある「**価格**」セクションを見る
   - **¥2,980** の行の右側に `price_1...` で始まる文字列がある
   - その文字列をコピー（クリップボードアイコンを押せばコピーされる）

---

## ステップ3：PHPファイルを開いて2箇所を書き換える

1. Finderで次のファイルを開く（テキストエディタ・VS Code・メモ帳でOK）：
   `/Users/kazuhiroakutsu/claude-seminar-lp/20260522b-s/v2/stripe-php/create-checkout-session.php`

2. 以下の **2箇所** を書き換える：

   **A. 11行目（Secret Key）**
   ```
   $stripe_sk = '__STRIPE_LIVE_SK__';
   ```
   ↓ ステップ1でコピーした `sk_live_...` を貼る
   ```
   $stripe_sk = 'sk_live_51H83U6LEBuO6I3Wl...（コピーしたキー全体）';
   ```

   **B. 15行目（Price ID）**
   ```
   $price_id  = '__STRIPE_PRICE_ID__';
   ```
   ↓ ステップ2でコピーした `price_1...` を貼る
   ```
   $price_id  = 'price_1XXXXXXXXXXXXXXXX';
   ```

3. 保存（Ctrl+S / Cmd+S）して閉じる

---

## ステップ4：FTPソフトで Xserver にアップロード

1. FileZilla（または使い慣れたFTPソフト）を起動

2. 接続情報を入力：
   - **ホスト**：`sv6262.xserver.jp`
   - **ユーザー**：`xs161634`
   - **パスワード**：阿久津さんが管理している Xserver FTP のパスワード
   - **ポート**：21

3. 接続したら、**右側（サーバー側）** で次のフォルダに移動：
   `/fp-1.info/public_html/`

4. その中に「**stripe**」というフォルダを**新規作成**する（右クリック→ディレクトリの作成）

5. 作った `stripe` フォルダの中に入る

6. **左側（自分のMac）** で次のファイルを探す：
   `/Users/kazuhiroakutsu/claude-seminar-lp/20260522b-s/v2/stripe-php/create-checkout-session.php`

7. そのファイルを**右側にドラッグ&ドロップ**してアップロード

## ステップ5：動作確認

1. ブラウザで次のURLを開く：
   https://lp.well-c.biz/20260522b-s/v2/#cta

2. 「アーカイブ動画＋PDF資料」のカード部分が、**Stripeのクレジットカード入力フォーム（カード番号・有効期限・CVCの入力欄が直接表示される画面）に切り替わっていれば成功**

3. もしまだ「クレジットカードで購入する（¥2,980）」ボタンの旧カードが見えていたら、ブラウザを Ctrl+Shift+R（Windows）または Cmd+Shift+R（Mac）で強制再読み込み

---

## うまくいかないとき

- カード入力欄が出ない → ブラウザのアドレスバーで次のURLを直接開いて、何が返るか確認：
  https://fp-1.info/stripe/create-checkout-session.php
  - 「`{"error":"stripe_error",...}`」と出る → Secret Key か Price ID の貼り間違い
  - 「404 Not Found」 → ファイルの置き場所が間違っている（`/public_html/stripe/` の中に置く）
  - 「500 Internal Server Error」 → ステップ3の書き換えがうまくいっていない

このURLをそのまま私（Claude）に貼って教えてくれれば、原因を特定して直します。
