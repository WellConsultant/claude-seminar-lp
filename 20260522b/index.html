DOES THIS SOUND FAMILIAR?

こんな状態に、心当たりはありませんか？

一つでも当てはまるなら、この制度を知る意味があります。

やりたい事業はあるのに、日々の業務に追われ、構想のまま止まっている
補助金や支援制度を調べても、自社・顧問先でどう使うか判断できない
事業計画書を作ったことはあるが、社内や現場が動かなかった
良い商品・サービスのはずなのに、顧客へ価値が伝わらず売上につながらない
人手不足や人材育成の課題で、新たな取り組みに踏み出せない
顧問先に、単なる制度紹介ではない“次の一手”を提案したい

THE REAL PROBLEM

動かない原因は、
努力不足ではありません。

補助金を取るため。金融機関へ説明するため。社内で否定されないため。失敗したときに責められないため。

そうして作られた計画は、きれいにまとまって見えます。しかし、経営者自身が本当に進みたい方向や、現場で実行するための障害が整理されていなければ、計画は棚に置かれたままになります。

本当は、新しい事業を形にしたい。顧客にもっと価値を届けたい。従業員が成長できる会社にしたい。顧問先をもっと深く支援したい。

その思いがあるのに、
また「提出するためだけの計画」を
作って終わってしまう。
今回の勉強会で扱うのは、
本当に動かしたい事業を、現実的な計画に落とすための制度活用です。

ABOUT THE PROGRAM

今回扱う制度について

早期経営改善計画策定支援（バリューアップ支援事業）は、
認定経営革新等支援機関の支援を受けながら、早い段階で経営改善へ着手するための制度です。

POINT 01

現状と課題を整理する

資金繰りや採算、事業の構造、改善すべき課題を把握し、何から取り組むべきかを明らかにします。

POINT 02

動ける計画に落とし込む

資金計画・ビジネスモデル俯瞰図・アクションプラン等を通じて、経営改善の道筋を具体化します。

POINT 03

伴走支援まで対象になり得る

計画策定後も、実行状況を確認しながら進める伴走支援が制度設計に含まれています。

POINT 04

費用の2/3・上限80万円

計画策定支援費用と伴走支援費用について、所定の要件を満たす場合、費用の2/3・上限80万円以内が支援対象となります。

重要：最大80万円は、勉強会参加費への補助ではありません。制度要件を満たす事業者が、認定経営革新等支援機関による計画策定支援・伴走支援を受ける際の対象費用について、所定の手続により支援を受け得る上限額です。
