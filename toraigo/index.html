FOR HR / 管理担当者の方へこんなお悩みはありませんか？

人材育成・社内研修を担当されている方からよくいただくご相談です。

研修をやっても「受講したか」が分からない

「動画リンクを配って終わり」になり、誰が・どこまで・本当に視聴したかを後から確認する手段がない。受講者が早送り・スキップ視聴している可能性も否定できない。

助成金監査で「視聴の証跡」を求められた

人材開発支援助成金の事業リスキリングコース等を申請したが、視聴ログ・進捗証跡を客観的な記録として提出できる仕組みがない。Excel手集計では監査時に再現性が示せない。

会社が増えると管理が指数関数的に大変になる

グループ会社・取引先・販社など、研修対象会社が増えるたびに「誰が誰の研修を見る権限を持つか」「会社ごとの進捗をどう見せるか」が複雑になり、Excelとメールでは破綻する。

受講者にとっての画面が分かりにくい

「自分が今どのコースをどこまで進めているのか」「次に何を見ればよいのか」がパッと分からないと、受講率は伸びない。結果、研修への投資が回収できなくなる。

CORE FEATURESTraingoの主な機能

受講者・企業管理者・サービス全体の管理者、それぞれに最適化された画面と権限を1つの製品で提供します。

A

視聴ログ自動記録・スキップ禁止

動画は秒単位で視聴記録を残し、早送りやスキップは規定上止められる設計。助成金監査で求められる「実視聴時間」「最終視聴日時」「修了日」を、人手介入なしの証跡として保存します。

  • watch_logs テーブルに秒単位記録
  • 視聴開始日時・最終視聴日時・進捗率を自動算出
  • 監査向けCSV一括ダウンロード機能つき
B

会社ごとに独立したスクール（研修空間）

1つの法人で複数のスクール（業種別・部門別・販社別）を作成し、それぞれを独立した研修空間として運用できます。担当者は自社のスクール内だけを操作でき、他社の情報には触れません。

  • 会社ごと独立のスクール／コース／受講者管理
  • 権限により他社情報は物理的に閲覧不可
  • 1コースを複数会社へ同時割当も可能
C

章・レッスン・クイズの3階層コース構造

コースを「章 → レッスン（動画／文章／クイズ）」に整理。修了判定は必須レッスン全完了で自動判定し、修了証PDFを自動発行します（オプション）。受講者は今どこにいるかが一目で分かります。

  • 章・レッスンの並び替え／追加／削除
  • 動画レッスン・文章レッスン・クイズの3種類
  • 必須／任意の指定で柔軟な修了条件設計
D

受講者のアカウント自己管理

受講者は自分のプロフィール（氏名・メール・パスワード）・購入履歴・問い合わせ送信までを1画面で完結。担当者の代行作業を最小化します。

  • プロフィール／パスワード変更
  • 受講中コースカード・進捗バー表示
  • スクール管理者への問い合わせ送信機能
E

受講者の自動登録（オプション）

クレジット決済（Stripe連携）と組み合わせることで、研修コースの販売リンクで購入された方を、自動で受講者として登録＋ログイン情報をメール送信する運用が可能です。

  • Stripe Payment Link 自動発行
  • 決済完了Webhookで自動受講者登録
  • ランダムパスワード生成＋招待メール送信
F

助成金監査用CSV一括出力

受講者ごとの「視聴開始日時／実視聴時間／修了日／進捗率」を1ファイルでまとめてダウンロード。提出書類の作成時間を大幅短縮します。

  • 全受講者・全動画の視聴記録を1ファイルに集約
  • 監査担当者にそのまま提示できる形式
  • 期間指定・コース指定での絞り込み出力にも対応

3 ROLES3つのロールに最適化

受講者・企業管理者・サービス全体管理者、それぞれの立場で必要な画面と権限を最初から内蔵しています。

受講者

学習に集中できる画面

「自分が今どこで何をすべきか」が迷わない設計。続きから学ぶ／マイライブラリ／修了済み、の3区分。

  • コースカード一覧
  • 章・レッスン階層表示
  • 進捗バー／状態バッジ
  • プロフィール自己管理
企業管理者

自社の研修を運営する画面

自社のスクール作成、コース作成、受講者の招待、視聴進捗の確認まで、自社の範囲内で完結します。

  • スクール管理
  • コース・動画管理
  • 章・レッスン編集
  • 受講者の進捗確認
管理者

サービス全体の管理画面

会社・コース・受講者を横断的に管理。コースを特定の会社に複数割当する権限も持ちます。

  • 全会社・全コース一覧
  • コース→会社の割当（複数可）
  • 全受講者の進捗監査
  • 監査用CSV一括出力
